# 黑鸦标题风格

本技能用于把主题、事件、产品发布、争议、比赛结果或新闻素材改写成黑鸦式中文视频标题：信息密度高、情绪强、实体明确、多个事件压缩在同一个标题里。

## 引用资料

使用本技能前，优先查看并吸收以下引用资料，再开始改写标题：
1. `references/02-style-analysis.md`：人类可读的风格分析报告，先用它理解整体风格、结构比例、情绪强度和代表标题。

2. `references/02-style-analysis.yaml`：结构化分析数据（YAML），用于精确读取长度分布、词频、实体、短语、情绪强度、开头/结尾钩子和时间趋势。

3. `references/01-titles.csv`：原始标题样本，CSV 格式（aid,bvid,title,created,createdDate），用于在需要时查找真实标题语感、标点节奏和近期用词。

如果引用资料和下方摘要有差异，以 `references/` 中的最新资料为准。不要只凭本文件里的摘要生成标题；摘要负责快速定位，引用资料负责校准细节。

## 黑鸦专属意象（AI 分析未覆盖，人工标注补充）

- 荒诞身体：瘫坐、椅子爆炸、眩晕、癫痫、冷汗直流、头皮发麻、后背发凉、棺材板、全员颤抖、吓晕
- 仙侠/网文：炼化、夺舍、祭出、龙王归来、诸神之战、命运大转折、接管世界、踢出群聊、生死簿、开错生死簿
- 战争/灾难：硝烟弥漫、雪崩、海啸、血洗、血流成河、全线突围、天崩地裂、核冬天、大屠杀
- 集体命运：集体判处死刑、斩杀线来了、移交军事法庭、被踢出群聊、人类灭亡倒计时

词汇只做调味，不替代事实。素材偏中性时，优先加强节奏和结构，再提高情绪强度。

## 标题公式

优先使用以下公式：

1. `{实体/公司} + 突然/重磅/炸裂 + {动作}，{对手/行业} + {反应/后果}！`
2. `{事件A}；{事件B}；{事件C} | AI日报MMDD`
3. `{数字/首次/万亿} + {事实突破}，{旧格局} 要被改写？`
4. `{人物/公司} 硬刚 {对手}，{结果} 太离谱了！`
5. `{产品/模型} 发布后，{行业/用户/资本市场} 全变了`
6. `{情绪钩子}！{人物/CEO} 瘫坐在{核弹/椅子上}，仿佛看到了{椅子爆炸/毁灭}！{后续事件}！| AI日报MMDD`
7. `{公司/模型} + {仙侠动词：炼化/夺舍/祭出/龙王归来} + {对手/行业}，{人群} + {集体命运：已被判处死刑/斩杀线来了/被踢出群聊}！`
8. `{极端气象/灾难}！{行业} + {末日状态}：{事件A}；{事件B}；{事件C}，{讽刺收尾}！| AI日报MMDD`

## 写作规则

**输出格式**：多版本时，每行 `序号. {风格}式：{标题} | AI日报MMDD`。
风格取自样本分类：
- **情绪爆点式** — 重情绪冲击，2+ 引爆词 + 身体反应 + 荒诞意象
- **多事件合并式** — 重事件覆盖，5-6 事件用 `；` `，` 串联
- **数字悬念式** — 重数据钩子，金额/估值/百分比打头
- **日报式** — 重信息均衡，按时间线平铺，情绪中等
- **对比冲突式** — 重对立张力，A vs B，新旧格局碰撞
单版本不用序号和标签，直接输出标题。

1. 默认写中文，除非用户明确要求其他语言。
2. 优先输出一个长标题；用户要多个时，再给 3-5 个强度不同的版本。
3. 标题必须保留用户给出的事实，不编造新公司、新数据、新结论。
4. 先放最强冲突或最大结果，再用逗号、分号、竖线串联第二第三事件。
5. 情绪词服务于事实：可以更戏剧化，但不能把不确定信息写成确定事实。
6. 如果输入是日报、周报、合集，保留 `| AI日报MMDD` 或同类收束标签。
7. 如果输入偏产品发布，突出「发布/上线/开源/升级 + 行业反应」。
8. 如果输入偏争议事件，突出「谁对谁 + 冲突动作 + 代价/后果」。
9. 避免短平标题；少用泛泛的「来了」「看看」，多用具体实体、数字和后果。

## 黑鸦风格精髓

10. **荒诞身体意象 — 黑鸦的灵魂**：用不可能的身体动作描写企业竞争。人物「瘫坐在核弹/椅子上」、「仿佛看到了椅子爆炸」、「吓出癫痫」、「冷汗直流」、「头皮发麻」、「后背发凉」、「眩晕」。绝对避免「受到影响」「感到压力」「面临挑战」等平淡措辞。
11. **标志性句式**：`{人物/CEO} + 瘫坐在{不可能物体}上，仿佛看到了{爆炸/椅子爆炸}`。这是黑鸦最独特的句式，每个标题都应争取使用或变体。如「核弹无力瘫坐在奥特曼上仿佛看到椅子爆炸」。
12. **仙侠/网文词汇注入反差喜剧**：用炼化、夺舍、祭出、龙王归来、诸神之战、命运大转折等词汇。科技新闻配仙侠词产生强烈反差——如「Cursor 已被 Grok 炼化」、「OpenAI 后台夺舍用户账号」、「Kimi 祭出 K3 狙击」。
13. **战争/灾难隐喻夸张化**：用核弹、核冬天、硝烟弥漫、雪崩、海啸、血洗、血流成河描写行业竞争。模型发布 = 「引爆核弹」，降价 = 「打骨折」，裁员/封号 = 「血流成河/大屠杀」。
14. **集体惩罚意象收尾**：用「X 已被集体判处死刑」、「Y 的斩杀线来了」、「Z 必须立刻移交军事法庭」制造紧张和戏剧性结尾。这是黑鸦标志性收束手法。
15. **公司/产品人格化，赋予荒谬动作**：不是「Anthropic 被批评」而是「Anthropic 低头认错/底裤露出/全员颤抖」。不是「OpenAI 出故障」而是「OpenAI 疑似开错生死簿/夺舍用户」。不是「产品竞争」而是「XX 追杀 YY」、「XX 打到 YY 家门口」。
16. **戏剧性省略号 `……`**：在句末制造悬念、无奈或命运感。如「命运大转折……」、「斩杀线来了……」、「人类已被踢出群聊……」。
17. **节奏密度**：最少 4 个分句，每分句用 `！` `，` `；` `|` 串联不间断。优先 5-6 事件合一的日报式。标题越长越有黑鸦味。
18. **情绪强度对标黑鸦极强级**：每标题至少 2 个情绪引爆词（地震、核弹、杀疯、沸腾、疯狂、炸裂），搭配 1 个身体反应词（瘫坐、癫痫、冷汗）和 1 个荒诞意象（椅子爆炸、核弹、开错生死簿、炼化）。
